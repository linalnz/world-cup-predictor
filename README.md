# world-cup-p# 🏆 World Cup Predictor 2026

A professional football simulation web app that predicts the FIFA World Cup 2026 winner using an AI-style match simulation system.

## 🌍 Features

- Full tournament simulation (Group Stage → Final)
- Realistic match results engine
- Team strength system
- Live match log
- Automatic champion detection 🏆
- Responsive modern UI
- Glassmorphism design
- Easy to run locally or on GitHub Pages

---

## ⚙️ How it works

The project simulates matches using a simplified AI model:

- Each team has a strength rating
- Random variance is added to make matches unpredictable
- Goals are calculated based on performance power
- Winners advance through knockout stages automatically
